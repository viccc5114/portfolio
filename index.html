Projects

Project 1: GBDA 101 – Icons

This project explores Chinese traditional instruments through a set of custom icons. Each instrument is simplified into clear shapes and textures while keeping its unique character. The goal was to balance cultural references with a modern, playful visual style that could be used in digital interfaces or printed media.

Next Project

Project 2: GBDA 228 – Website Redesign

For this project I redesigned an HVAC engineering website to create a more modern, user-centered experience. I focused on improving visual hierarchy, readability, and navigation. The redesign introduces a clearer layout, stronger typography, and more consistent imagery so clients can quickly understand services, projects, and contact information. Content is reorganized into intuitive sections to make the site easier to scan and explore.

You can compare this project with the original company website here: Original HVAC Site.

Project 3: GBDA 201 – Documentary

This documentary explores how people use the MBTI personality types in their everyday lives and relationships. Through interviews, surveys, and narration, we look at how the four-letter types can help people reflect on their emotions, habits, and communication styles— while also questioning the limits and misconceptions of MBTI.
